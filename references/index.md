# 参考文献库

> 统一引用数据库，所有研究引用的来源都在这里

## 引用格式说明

本系统采用 GB/T 7714 格式（学术标准）：

- 书籍：`作者. 书名[M]. 出版社, 年份.`
- 论文：`作者. 标题[J]. 期刊, 年份.`
- 报告：`机构. 标题[R]. 地点: 机构, 年份.`
- 政策：`部门. 文件名[EB/OL]. 年份.`

## 引用ID规则

格式：`[类型][年份]`

例如：
- `mof2023` = 财政部2023年政策
- `iea2025` = IEA2025年报告
- `moore1993` = Moore1993年论文

