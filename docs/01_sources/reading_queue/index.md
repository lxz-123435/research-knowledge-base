## 阅读建议

### 研究效率原则

1. **相关性优先** - 先读与研究主题直接相关的
2. **经典文献优先** - 先读奠基性论文
3. **权威报告优先** - 先读IEA/OECD/世界银行等权威机构
4. **新旧结合** - 经典理论+最新研究

### 阅读节奏

| 类型 | 建议时间 |
|------|----------|
| 书籍 | 1-2周/本 |
| 论文 | 2-3天/篇 |
| 报告 | 1-2天/份 |
