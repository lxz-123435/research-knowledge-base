# 思想通信 | 如何认识对外开放从“跟跑”走向“领跑”

\[学习时报]\(javascript:void$0$;) *2025年12月1日 18:02* *北京*

![](https://mmbiz.qpic.cn/sz_mmbiz_png/6xz9HxNXZQSX7QyYX5LviaUlsRF4QSOPswmqkurLsbwNKIV833P6mg363Qicc2bcNFhfSG6WgfaOw83l8s1OvojA/640?wx_fmt=png\&tp=wxpic\&wxfrom=5\&wx_lazy=1#imgIndex=0)

**提问：谢春 湖南省退役军人事务厅党组书记、厅长**

习近平总书记指出，过去我们的开放是“跟跑”，现在我们的开放是要“领跑”，这是更高水平的开放。请问，如何理解这一论断?

**解读：陈建奇 中央党校（国家行政学院）国际战略研究院副院长、教授**

坚持开放合作、互利共赢是中国式现代化的必然要求。习近平总书记强调，过去我们的开放是“跟跑”，现在我们的开放是要“领跑”，这是更高水平的开放。从“跟跑”到“领跑”，深刻揭示新时代高水平对外开放的新使命新任务，也体现了中国的大国担当。党的二十届四中全会通过的《中共中央关于制定国民经济和社会发展第十五个五年规划的建议》强调，扩大高水平对外开放，开创合作共赢新局面。稳步扩大制度型开放，维护多边贸易体制，拓展国际循环，以开放促改革促发展，与世界各国共享机遇、共同发展。全面释放了未来五年乃至更长时期持续推进高水平对外开放的行动自觉，更深刻指明了引领塑造世界开放新格局的路径选择。

践行共商共建共享的全球治理观，引领普惠包容的经济全球化

当前，世界百年未有之大变局加速演进，逆全球化思潮蔓延，贸易保护主义、单边主义升温，地缘政治冲突问题突出，国际环境复杂严峻，世界开放合作呈现新特点新挑战。国际合作从侧重效率转向更加关注公平，有些国家不是聚焦做大经济蛋糕，而是更加重视分蛋糕，促使开放面临着零和博弈的挑战。世界从侧重发展转向更加重视安全，能源安全、粮食安全及产业链供应链安全备受关注，经济问题泛政治化泛安全化趋势明显。中国始终坚持顺应全球化发展大势，积极践行共商共建共享的全球治理观，倡导平等有序的世界多极化和普惠包容的经济全球化，推动构建人类命运共同体，为世界开放合作贡献中国新理念。

坚持合作共赢新理念。相互依存是全球化的重要特征，世界形成了你中有我、我中有你的局面，世界各国命运与共。人为把世界各国分成三六九等，主张强者主导世界、弱者被动接受的方式，不仅无助于全球的发展，而且让世界遭受人为的割裂，世界冲突摩擦风险加剧。习近平总书记强调，国际社会发展到今天已经成为一部复杂精巧、有机一体的机器，拆掉一个零部件就会使整个机器运转面临严重困难，被拆的人会受损，拆的人也会受损。中国倡导合作共赢、共同发展理念，秉持和而不同，国家不论大小、强弱，不论采取何种制度模式，都要相互尊重、互利共赢，让团结代替分裂、合作代替对抗，促进优势互补，兼顾自身利益和他方利益，共同做大世界经济蛋糕，形成各国相互成就、共同发展的合作格局。

坚持真正的多边主义。金融危机以来世界经济增长分化，叠加新冠疫情的冲击，全球经济增长不确定性不稳定性问题突出。面对风高浪急、惊涛骇浪的巨大考验，世界各国必须携手合作、同舟共济、共克时艰。但单边主义与阵营对抗正在侵蚀多边体系根基，搞排他性“小圈子”，动辄毁约退群、拉帮结派，背离多边主义的单边行径只会加剧分裂与动荡。中国始终是多边主义的坚定践行者和捍卫者，通过中非合作论坛、上合组织峰会、金砖峰会及“一带一路”国际合作高峰论坛等扩大朋友圈，通过APEC峰会、G20峰会等共同探寻世界经济发展新方案。中国提出全球发展倡议、全球安全倡议、全球文明倡议、全球治理倡议，推动构建人类命运共同体，以务实行动诠释多边主义内涵，为推动世界开放合作夯实治理根基。

坚持普惠包容的经济全球化。经济全球化是大势所趋，全球化推动世界经济较快发展，但也出现了发展不平衡的问题，那些没有得到好处的国家、地区或者群体，出现了民粹主义、民族主义，构成全球开放合作的巨大阻碍。唯有让不同国家、不同阶层共享发展成果，全球化才能行稳致远。中国坚持推动普惠包容的经济全球化，从推动金砖国家扩容、上合组织扩员，到落实全球发展倡议，实施数千个民生项目;积极通过缓债、技术转让、能力建设等方式，推动后发国家补短板强优势，解决发展瓶颈，推动联合国2030年可持续发展议程。中国用实际行动诠释着“发展是世界各国的权利，而不是少数国家的专利”，让经济全球化真正成为一条惠及世界各国人民的繁荣之路。

推进制度型开放，引领制定高标准国际经贸规则

当前世界贸易组织改革进程受阻，争端解决机制难回正轨，背离全球治理体系搞双重标准、长臂管辖屡见不鲜，国际经贸规则标准遭受严重挑战。中国坚定维护以世界贸易组织为核心的多边贸易体制，积极扩大自主开放，实施自由贸易试验区提升战略，稳步扩大规则、规制、管理、标准等制度型开放，对接高标准国际经贸规则，探索数字经济、人工智能等新规则新标准，参与引领新规则新标准的制定。

对标国际高标准经贸规则。当前国际经贸规则正经历新一轮重塑，以《全面与进步跨太平洋伙伴关系协定》(CPTPP)、《区域全面经济伙伴关系协定》(RCEP)、《数字经济伙伴关系协定》(DEPA)等为代表的高标准国际经贸协定，正推动国际规则从关税等边境规则向劳工标准、知识产权、环境保护、政府采购等边境后规则拓展。中国积极对接高标准国际经贸规则，积极加入并推动RCEP发展，积极申请加入CPTPP和DEPA，不断完善相关法律法规，提升了我国营商环境的国际竞争力。构建与国际通行规则相衔接的制度体系，向世界展示中国参与全球规则制定的决心和能力，把中国大市场主动融入全球高水平开放体系，既推动世界整体开放水平的提升，也让世界分享中国发展的重大机遇，引领了高水平开放市场提质扩围。

自主探索高水平开放新体制。中国已经设立22个自由贸易试验区，海南正在建设自由贸易港，积极探索高水平开放新体制。实施负面清单管理制度，外商投资准入负面清单从最初的190项减至2024年版外商投资准入特别管理措施(负面清单)的29项，负面清单制造业条目“清零”，海南推出第一张跨境服务贸易负面清单，推动服务贸易与货物贸易自由化水平稳步提升。自贸试验区持续推动改革试点，形成了以开放促改革、以改革促发展的重要机制。国际贸易“单一窗口”持续扩围升级，实现海量货物快速通关便利化与高效率监管双提升的目标，展现了自主开放的灵活性与稳健性。积极推进人民币国际化，扩大资本项目开放程度，建设自主可控的人民币跨境支付体系，稳步发展数字人民币，持续提升金融开放水平。中国对同我国建交的最不发达国家实施100%税目零关税，对48个国家实行单方面免签政策，彰显对外开放的行动自觉。

推动新兴领域开放。新一轮科技革命和产业变革深入发展，信息通信、大数据、人工智能等技术广泛应用，催生出新业态新模式。中国在新兴领域坚持开放与治理并重，既积极扩大开放合作，又注重构建安全可控的治理体系。在数字经济领域，中国出台《数据出境安全评估办法》《促进和规范数据跨境流动规定》等法规，探索构建数据开放治理新机制。在人工智能领域，中国提出《全球人工智能治理倡议》等方案，倡导构建开放、公正、有效的治理机制，形成具有广泛共识的人工智能治理框架和标准规范，促进人工智能技术造福于人类。在发展空间方面，中国提出“数字丝绸之路”倡议，加强在数字经济、人工智能、纳米技术、量子计算机等前沿领域合作，推动大数据、云计算、智慧城市建设，连接成21世纪的“数字丝绸之路”。

推动共同开放，引领建设开放型世界经济

“一枝独秀不是春，百花齐放春满园”。开放不是单个国家的“独角戏”，而是世界各国的“大合唱”。中国在扩大高水平对外开放的同时，积极推动世界各国共同开放，引领建设开放型世界经济，更好惠及各国人民。

维护产业链供应链稳定。习近平总书记强调，维护全球产业链供应链韧性和稳定是推动世界经济发展的重要保障，符合世界各国人民共同利益。中国积极构建关键物资储备库，加大对能源资源及芯片等关键物资的储备，保障生产持续稳定健康的运行，为世界各国源源不断提供中间品和重要原材料，发挥世界制造业中心稳定全球产业链供应链的巨大效能。与此同时，中国积极推动开放式创新，加快集聚人才、资本、管理、技术等创新要素，推动全球产业建链延链补链强链，坚定不移维护产业链供应链的公共产品属性，保障世界生产发展安全稳定，以实际行动深化产业链供应链国际合作，切实保障世界各国生产生活的需要。

建设国际合作新平台。中国通过打造多层次、宽领域的国际合作平台，形成“展会搭台、产业唱戏、规则共建”的开放生态。中国设立了进博会、广交会、服贸会、消博会等具有全球影响力的展会平台，不仅成为世界商品服务开放的窗口，也成为世界技术交流、理念碰撞的平台。在进博会上，世界前沿技术与创新产品向全球开放共享。在服贸会上，数字服务、绿色低碳等新兴领域引领开放合作新趋势。通过创新国际合作平台，吸引全球优质资源集聚，形成“引进来”与“走出去”双向互动的产业生态。外资企业依托中国产业集群降低生产成本、拓展市场空间，中国企业通过平台对接全球先进技术与管理经验，加速转型升级。

高质量共建“一带一路”。以“硬联通”夯实开放型世界经济的基础设施根基。中老铁路、雅万高铁、比雷埃夫斯港等标志性项目稳步运营，中欧班列累计开行量已突破11万列，打通了区域互联互通的“堵点”，让商品及各类生产要素得以在更大范围内高效流动。以“软联通”完善开放型世界经济的规则体系。基于平等协商的规则共建，摒弃了霸权主义的规则垄断，让开放型世界经济的制度框架更具包容性与公平性。以“心联通”凝聚开放型世界经济的合作共识。“一带一路”合作增进民心相通，超越单纯的经济范畴，成为各国人民增进理解、凝聚信任的纽带，增强了相关国家民众对经济发展和生活改善的获得感，开放型世界经济发展有了更为坚实的民意基础。
