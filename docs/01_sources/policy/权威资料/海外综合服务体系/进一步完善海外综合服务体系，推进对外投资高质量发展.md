进一步完善海外综合服务体系，推进对外投资高质量发展罗雨泽

国务院发展研究中心对外经济研究部副部长、一级巡视员

10月14日，经国务院同意，商务部等5部门联合印发《关于进一步完善海外综合服务体系的指导意见》（以下简称《指导意见》），是我国创新完善“走出去”制度与政策体系的重要举措，具有标志性意义。《指导意见》作为我国首个海外综合服务领域的指导文件，将有力推动我国海外综合服务体系的建设与完善，在提升出海企业国际竞争力、维护我海外正当权益、推进我国对外投资高质量发展等方面发挥有力的支撑、保障和促进作用。

# 一、进一步完善海外综合服务体系是服务高水平“走出去”的必然要求

2024年底，中央经济工作会议提出要“完善海外综合服务体系”，这是综合考虑我国对外投资合作进入新阶段和国际形势发生重大变化做出的重要部署安排。

我国已经成为全球对外投资大国，海外利益分布广泛。截至2024年底，我国对外直接投资存量达3.14万亿美元，连续8年居全球前三位，流量连续13年居全球前三位，占全球份额连续9年超过一成，在境外设立企业5.2万家，覆盖全球超过80%的国家和地区，对外资产总规模高达10.2万亿美元。在全球跨境投资低迷的情况下，得益于我国企业竞争力的提升、全球产业布局的需要以及中国开放包容的国际合作战略，我国对外投资保持稳定增长，展望未来，仍有很大潜力和空间。

国际投资环境日益复杂，海外利益保护的紧迫性增强。近年来，百年未有之大变局加速演进，世界经济增长放缓，经济全球化遭遇“逆风逆流”，国际环境日趋复杂。出海企业面临的海外经营风险挑战不断增多，需要着力构建与我国对外投资贸易发展相适应的海外综合服务体系。我海外综合服务发展相对滞后，存在明显短板。一方面，随着我对外投资规模的增加、覆盖面扩大以及国际环境的变化，对综合服务的需求显著增加，另一方面，法律、金融、物流等专业服务业走出去相对滞后，服务平台也存在层级不高、功能不全的问题，成为企业提升海外投资效率和效益的短板和痛点。例如，不同国家的法律体系、司法制度和实践差异较大，由于很多出海企业、尤其是中小企业对东道国法律环境缺乏了解，本土法律专业服务机构提供海外法律服务的能力相对薄弱，出海企业面临高昂的成本和风险。

# 二、《指导意见》是完善海外综合服务体系的顶层设计和操作指南

《指导意见》完整准确全面贯彻新发展理念，围绕加快构建新发展格局，坚持统筹发展和安全，基于企业出海全周期、全链条、全方位分析，提出了6方面16项具体举措。这些举措目标明确、分工清晰，且相关关联，体现出央地上下、区域跨部门左右、企业自身和公共服务内外等多向贯通、多维协调的特点，体现了有为政府与有效市场的结合，表现出“一高三强”特征：

一是层级高。《指导意见》明确了在国家层面需要做的重点工作，比如，提出将走出去公共服务平台升级为国家层面海外综合服务平台，鼓励支持全国性行业商协会发挥协调作用，利用好现有国家级展会平台、丰富双边合作机制等。

二是综合性强。如，国家层面建设的海外综合服务平台，整合统筹外事、法律、财税、金融、经贸、物流、海关、贸促等领域服务资源，贯穿企业出海全流程、全链条，联通“税路通”、“贸法通”、条约数据库、国家海外知识产权信息服务等平台资源，提升平台集成化、综合化、数智化水平，形成综合服务大平台和专业服务分平台有机结合的“1+N”服务体系；地方服务集成，包括“线上+线下”服务渠道建设，打造企业出海获取基础性信息的“一站式”窗口，持续优化相关业务办理流程，集聚法律服务、知识产权、财务会计等领域优质专业服务机构。

三是针对性强。如针对企业出海存在的风险问题，从风险跟踪、评估和应对等多方面提出营造安全有利环境；针对投资贸易争议，提出用好多元化争端解决机制，增强应对贸易摩擦的能力；针对高质量、专业化服务短板，提出了提升法律、财会、咨询、金融等各领域服务水平的具体举措；针对一些企业海外合规能力短板，提出了增强企业跨国经营能力、加强国际规则与理念衔接、提升海外经营合规水平的具体意见。

四是可操作性强。《指导意见》给出了具体的工作抓手，比如在优化企业“走出去”业务流程时，提出要依托地方政务服务中心；在探索创新服务模式时，提到了依托自由贸易试验区、自由贸易港、国家服务业扩大开放综合示范区等开放平台建设出海综合服务港；在延伸健全海外服务时，明确了以境外经贸合作区和海外综合服务站为切入点；在发挥特定地区作用方面，提到了港澳地区在跨境投资贸易、金融、法律等领域优势。

# 三、下一步需抓好落实工作

一是制定操作细则、加强宣传解读。《指导意见》涉及多个层面、诸多主体，有关地方可在此基础上制定细化实施方案，进一步分解任务、明确分工，加强政策宣传解读，让相关主体知悉、知用。

二是加强现有经验分享。目前，有些地方探索已取得成效、积累了一定经验。如，浙江从“一个平台、一窗受理、建强网络、加强服务、强化监测”五大维度提出18项具体服务举措，为企业合理有序开展境外投资提供支撑；上海搭建的虹桥海外发展服务中心聚集了30多家专业服务机构，覆盖了金融、法律、仲裁、知识产权等业务功能；深圳开发“出海e站通”，围绕产品出海、投资出海、服务出海三大领域，建设企业出海综合服务体系。可做好经验总结、加强分享，在其他合适的地区进行复制推广。

三是做好跟踪评估、动态完善。完善海外综合服务体系是一项系统工程，也是一项需长期持续推进的工作。当前全球跨境投资面临的不稳定、不确定因素增多，我国日益走近世界舞台的中央，面临的国际局面更加复杂，形势变化可能会对完善海外综合服务体系提出新要求，需要及时总结经验，促进下一阶段完善工作。
