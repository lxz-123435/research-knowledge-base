# 欧盟-美国贸易和技术理事会

![](https://commission.europa.eu/sites/default/files/styles/oe_theme_medium_no_crop/public/2022-05/p037751-373326.jpg?itok=ul23jg_W)

欧盟和美国是坚定致力于推动数字化转型并基于共同的民主价值观（包括尊重人权）在新技术方面开展合作的合作伙伴。

欧盟-美国贸易与技术理事会是美国和欧盟协调应对关键全球贸易、经济和技术问题的平台，旨在深化基于共同价值观的跨大西洋贸易和经济关系。该理事会于2021年6月15日在布鲁塞尔举行的欧盟-美国峰会期间成立。

2024年4月4日至5日，欧盟和美国在比利时鲁汶举行了第六次 贸易和技术理事会[部长级会议。](https://ec.europa.eu/commission/presscorner/detail/en/ip_24_1827 "部长级会议。")
