# 分析模型库

> 可复用的分析模型

---

## 一、模型列表

### 1. PEST分析模型

政治（Political）、经济（Economic）、社会（Social）、技术（Technological）

适用场景：宏观环境分析

### 2. SWOT分析模型

优势（Strength）、劣势（Weakness）、机会（Opportunity）、威胁（Threat）

适用场景：战略分析

### 3. 波特五力模型

进入壁垒、替代品威胁、买方议价、卖方议价、现有竞争

适用场景：行业竞争分析

### 4. 价值链分析模型

基本活动、支持活动

适用场景：企业竞争优势分析

### 5. SCP分析模型

结构（Structure）、行为（Conduct）、绩效（Performance）

适用场景：产业组织分析

## 二、自有模型

### 1. 时代选择分析模型

```
时代背景 → 个人选择 → 价值判断 → 行动决策
```

来源：《法拉奇传》

### 2. 战地记者能力模型

```
采访技巧 → 抗压能力 → 语言能力 → 人脉网络
```

来源：《法拉奇传》
