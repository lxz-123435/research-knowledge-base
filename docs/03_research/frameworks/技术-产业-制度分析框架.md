# 技术—产业—制度分析框架

> 通用研究框架

---

## 一、框架概述

分析技术变革、产业演进与制度变迁之间的互动关系。

## 二、核心理论

- 技术创新理论
- 产业演化理论
- 制度经济学

## 三、三螺旋模型

```
    技术创新
        ↙   ↘
    产业演进 ←→ 制度变迁
        ↘   ↙
     市场需求
```

## 四、分析维度

| 维度 | 问题 |
|------|------|
| 技术层 | 技术成熟度、创新方向、替代风险 |
| 产业层 | 产业链位置、竞争格局、商业模式 |
| 制度层 | 监管政策、标准规范、法律框架 |

## 五、分析路径

1. 技术变革 → 产业机会 → 市场形成
2. 产业发展 → 制度需求 → 规则制定
3. 制度环境 → 产业约束 → 技术方向

## 六、适用场景

- 新兴产业研究
- 技术政策分析
- 产业转型研究

## 七、案例

- 人工智能产业发展
- 平台经济治理
