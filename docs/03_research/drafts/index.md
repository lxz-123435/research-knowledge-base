## 写作提示

### 如何从草稿到成品

1. **观点提炼** - 一句话说明白
2. **逻辑梳理** - 列出论证链条
3. **证据补充** - 找数据/案例
4. **框架应用** - 用现有框架分析
5. **成果转化** - 论文/报告/讲话

### 草稿发展路径

```
草稿 → 随笔 → 论文/报告
  ↓
洞见 → 框架 → 研究
```
