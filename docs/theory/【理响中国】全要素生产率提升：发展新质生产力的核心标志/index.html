    新质生产力是新时代条件下马克思生产力理论的最新发展和科学创新，实现了马克思主义政治经济学与历史唯物主义层面的双重跃迁。习近平总书记指出，新质生产力“以劳动者、劳动资料、劳动对象及其优化组合的跃升为基本内涵，以全要素生产率大幅提升为核心标志，特点是创新，关键在质优，本质是先进生产力。”我们必须“健全要素参与收入分配机制，激发劳动、知识、技术、管理、资本和数据等生产要素活力”。可以看到，“全要素生产率”已然成为我们理解新质生产力的一个高频词汇，它的提升构成了发展新质生产力的核心标志，而当下对这一重要概念的理解还不够全面。为此，需要厘清何为数字化时代的全要素生产率，辨明全要素生产率的三重特性，深刻把握全要素生产率赋能新质生产力的现实路径。 一、理解数字化时代的全要素生产率 数字化时代的全要素生产率着重于内部诸要素的优化配置、协调重组与功能互补。依传统观点来看，全要素生产率（Total Factor Productivity）也被称为技术进步率，是指在劳动、资本、土地等传统生产要素的数量和质量保持相对稳定、达到既定条件下，由于其他非传统要素的综合参与而达成的额外生产效率，具体表现为技术升级、政策改进、效率提高和规模效应等“余值效用”。在数字化时代，全要素生产率延续着这种“余值综合”的特质。但是，由于数据、算法、网络等数字化要素的更新迭代，完善全要素生产率不再仅仅体现为传统技术进步带来的生产效率提升，更在于实现其系统内部诸要素的优化配置、协调重组与功能互补。换言之，在数字化时代，数据等新型要素所发挥的智慧赋能和结构置换等效用，才是推进全要素生产率实现数字化转型、推动新质生产力发展的关键。 数字化时代的全要素生产率着重于数字技术、一般智力、政策环境等非物质要素的功效叠加与集成提升。数字化时代，全要素生产率既需要传统产业的有力支撑，更得益于数字经济和虚拟产业的高速发展，是衡量一个国家或地区经济发展潜力与国际竞争力的重要指标。在当今大力支持新质生产力发展的理论与现实背景之下，就是要予非物质要素以最大限度地认可和尊重，纵深推进产业数字化转型、虚拟技术的广泛应用，发挥数字经济对实体经济的赋能作用，推动我国技术密集型和知识密集型产业的双向发展，正确利用好以上非物质要素的叠加放大与集成提升功能。 数字化时代的全要素生产率是培育新质生产力的核心标志与强大动能。习近平总书记指出，发展新质生产力必须“坚持以我为主，从实际出发，大力推进自主创新、原始创新，打造新质生产力和新质战斗力增长极。要把握新兴领域交叉融合发展特征，加强集成创新和综合应用，推动形成多点突破、群体迸发的生动局面”。这点明了发展新质生产力的两个主要方向，一是要着重打造“三新”，即自主创新、原始创新和集成创新；二是要形成全国各地“个体性”和“总体性”并重的协调发展局面。全要素生产率能够成为培育新质生产力的核心标志与强大动能体现在：一方面，提升全要素生产率的关键本就在于增强本国的原始创新能力，减少对他国的技术依赖，形成跨界融合的技术集成创新；另一方面，全国各地也必须因地制宜、因变制宜地选择不同形式的全要素生产率提升路径，才能贴合各地区特殊情况形成要素集成合力。可见，在数字化时代，全要素生产率在发展新质生产力的全过程中扮演着“桥梁”和“基石”的重要角色，其“核心标志”与“强大动能”的基本属性必将在中国式现代化的实践中显现。 动态性。马克思指出：“手推磨产生的是封建主的社会，蒸汽磨产生的是工业资本家的社会。”这表明，生产力不仅体现为人对自然界的现实改造情况，更象征着人与人之间的社会关系，不同生产工具的变化恰恰显示出生产力形式和社会形式的新旧更迭。可以说，从传统生产力到新质生产力的演进和跃升，证明了生产力形式具备着动态发展的特征。全要素生产率的动态性体现在它会随着劳动、技术、资本、土地、数据、政策、环境等物质与非物质要素的不断变化而发生变化，需要我们密切关注和监测其整体结构，尤其注意其内部诸要素的位置调整、实时更新、功能演绎和新陈代谢。
