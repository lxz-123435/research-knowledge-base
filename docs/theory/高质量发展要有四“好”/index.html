    中国特色社会主义进入了新时代，我国经济发展也进入了新时代，基本特征就是我国经济已由高速增长阶段转向高质量发展阶段。高质量发展是能够更好满足人民日益增长的美好生活需要的发展，是将人民对美好生活的期盼变成现实的发展，是创新、协调、绿色、开放、共享的发展理念引领下的发展。 高质量发展要有“好的认识”。发展理念是发展行动的先导，是管全局、管根本、管方向、管长远的东西，是发展思路、发展方向、发展着力点的集中体现。“发展必须是科学发展，必须坚定不移贯彻创新、协调、绿色、开放、共享的发展理念。”新发展理念不是凭空得来的，是我们在深刻总结国内外发展经验教训的基础上形成的，也是在深刻分析国内外发展大势的基础上形成的，集中反映了我们党对经济社会发展规律认识的深化，也是针对我国发展中的突出矛盾和问题提出来的。新发展理念同时深刻揭示了我国未来发展的基本轨迹：走更高质量、更高效益、更加公平、更可持续的高质量发展道路。 高质量发展是体现新发展理念的发展，是创新成为发展第一动力、协调成为发展内生特点、绿色成为发展普遍形态、开放成为发展必由之路、共享成为发展根本目的的发展。高质量发展是经济发展从“有没有”转向“好不好”。新发展理念具有很强的战略性、纲领性、引领性，必须贯穿经济活动全过程。以新发展理念为引领，推动高质量发展，就是要把新发展理念作为指挥棒、红绿灯，对不适应、不适合甚至违背新发展理念的认识要立即调整，行为要坚决纠正，做法要彻底摒弃，真正做到崇尚创新、注重协调、倡导绿色、厚植开放、推进共享。 高质量发展要有“好的思维”。发展理念是否对头，从根本上决定着发展成效乃至成败。新发展理念科学回答了实现什么样的发展、怎样实现发展的问题，阐明了当前与长远、公平与效率、政府与市场、对内与对外、人与自然等重大关系。有效推进经济高质量发展，必须以依照新发展理念的整体性和关联性进行系统设计，树立全局观念和整体意识，以重点突破带动整体推进，在整体推进中实现重点突破。 高质量发展是一个系统性、完整性的全面发展，任何局部的、孤立的发展，都不可能推进真正体现新发展理念的高质量发展。我国正在开启现代化强国建设的新征程，作为指导我国发展实践的新发展理念，它体现了我国经济社会全面发展、现代化强国全面建设的本质要求，是一个涉及各个领域各项事业全面发展的系统化的发展理念。新发展理念是一个内涵十分丰富深刻的有机整体，涉及发展方向、发展目的、发展动力、发展主体、发展思路、发展重点、发展方式、发展环境和发展体制等各个领域。因此，新发展理念引领下的高质量发展是一个综合的系统性发展，有着丰富而深刻的内涵。就其“创新、协调、绿色、开放、共享发展”的内容范围所指，也早已超出了狭义上的“经济”范畴，而是经济社会的整体性发展。尽管发展的重心仍然是一般意义上的“经济”，但“新发展理念”显然是指导发展实践的发展观，具有鲜明的整体性和系统性。 高质量发展要有“好的预期”。现代市场经济发展的一个重要特点，就是发展信心、发展预期越来越成为经济发展的重要因素。高质量发展必须有“高的信心”“好的预期”。经济增长的内在动力，主要取决于企业家、投资者和消费者的信心及良好的预期。有了好的信心和预期，企业家才会去增加投资，消费者才会去增加消费，这样经济发展才能向好。经济发展预期和信心既是对国际国内整体发展前景综合判断的反映，也取决于经济、市场运行中宏观、中观政策环境的实际情形，还直接与当时面临这样那样的风险挑战相关。新发展理念指明了我国经济未来发展的方向、思路和重点，揭示了向好发展的趋向，符合经济社会发展的客观逻辑，加上我国改革开放40多年来的发展成就、独特的社会制度等优势，社会各界和客观公正的国际舆论对我国经济发展是充满信心的，总体预期也是积极健康的。 发展信心和发展预期是一种反映把握未来、塑造未来前景的治理能力。中国经济要稳增长，要推动高质量发展，既要依靠政府的计划目标和行政举措的驱动，更需要运用市场化的、法治化和改革开放的办法。这样才能保持根本性、常态性的信心和预期。高质量发展需要我们创造“好的预期”，处理好“不确定”与“确定性”的关系，要善于引导好预期，不断提高经济工作的预见性和确定性。 高质量发展要有“好的改革”。与新发展理念、高质量发展相适应，我们的体制机制政策应该更为科学合理配置。而体制机制的科学合理配置，就是要构建起系统完备、科学规范、运行有效的制度体系。为此，我们必须不断创新发展体制、重塑发展体制环境，“在解决发展动力，增强发展的整体性、协调性、平衡性、包容性等方面破难题、建机制，形成有利于创新发展、协调发展、绿色发展、开放发展、共享发展的体制机制”。 体制机制的改革创新，实际上就是经济资源有效配置、经济结构优化调整、经济运行方式的制度性安排问题。如何形成市场有效率、政府有作为、企业有活力的体制机制和宏观调控体系，如何通过体制机制的制度安排和政策配置来高质量、高效率地推动经济高质量发展？答案只能是：向改革开放要高质量的发展活力。现阶段改革的重点，就是要以供给侧结构性改革为主线，推动发展质量变革、效率变革、动力变革，提高全要素生产率，加快建设实体经济、科技创新、现代金融、人力资源协同发展的产业体系，构建市场机制有效、微观主体有活力、宏观调控有度的经济体制，不断增强我国经济发展的创新力和竞争力。以完善产权制度和要素市场化配置为重点，实现产权有效激励、要素自由流动、价格反应灵活、竞争公平有序、企业优胜劣汰。
