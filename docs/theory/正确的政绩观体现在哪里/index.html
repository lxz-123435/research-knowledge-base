    正确政绩观是为政之德、从政之道、施政之要，事关党和国家事业发展大局、事关人民群众切身利益、事关中国式现代化建设进程。中央经济工作会议强调，要树立和践行正确政绩观，坚持为人民出政绩、以实干出政绩，自觉按规律办事。这一重要论述为领导干部树立和践行正确政绩观指明了方向。 党的十八大以来，习近平总书记高度重视政绩观问题，围绕政绩为谁而树、树什么样的政绩、靠什么树政绩等发表一系列重要论述，身体力行、率先垂范，为党员干部树立了典范和标杆。广大党员干部深入学习贯彻习近平总书记的重要论述，把为民造福作为根本政治担当、最重要的政绩，真正把心思和精力放在为党和人民干事创业上，创造出了经得起实践、人民、历史检验的实绩。同时也必须清醒看到，在实际工作中，有少数党员干部重“口头”轻“实干”，喜欢作秀而不是做事，热衷于造势一时而不是造福一方，重显绩轻潜绩，不怕群众不满意、就怕领导不注意，只做面子、不做里子，只求即刻见效、不求利在长远，重“栽花”、轻“拔刺”，只吃“软蛋糕”、不啃“硬骨头”，喜欢掌声喝彩、害怕反对攻击，只愿与人为善皆大欢喜、不愿面对矛盾解决问题。这些不良政绩观会伤害群众感情、损害党的形象、影响党的事业，必须坚决予以纠正。 政绩体现在抓发展惠民生保稳定上。领导干部的价值在于为群众创造了多少价值，领导干部必须真正把心思和精力放在为党和人民干事创业上。新征程上，高质量发展是全面建设社会主义现代化国家的首要任务。要把坚持高质量发展作为政绩观的重要内容，完整准确全面贯彻新发展理念。人民幸福安康是推动高质量发展的最终目的，坚持为人民出政绩，要把抓发展和惠民生统一起来，把为老百姓做了多少好事实事作为检验政绩的重要标准，走好新时代党的群众路线，提高做群众工作的本领，用心用情用力解决群众急难愁盼问题，不断增强人民群众的获得感、幸福感、安全感。要坚持统筹发展和安全，做好防风险、保稳定各项工作，以高质量发展促进高水平安全，以高水平安全保障高质量发展。 政绩体现在实干上。业绩都是干出来的，真干才能真出业绩、出真业绩。积跬步以至千里，积细流以成江海。党的事业就是在一代又一代人的接续奋斗中推进的。1993年，时任福州市委书记的习近平同志亲自主持编制了“3820”战略工程，系统谋划了福州3年、8年、20年经济社会发展的目标、步骤、布局、重点等。当年提出的系统战略，在三十年如一日的坚持中已经转化为福州的发展现实。我们要发扬“功成不必在我”和“功成必定有我”的精神，既谋划长远，又干在当下，一方面，以只争朝夕的紧迫感、时不我待的责任心，积极作为、正确作为，做好当下能做的事情，坚决克服慢作为、不作为的现象；另一方面，对于长期战略任务，坚持一张蓝图绘到底、一茬接着一茬干，不贪一时之功、不图一己之私，坚决避免政绩冲动、盲目蛮干、大干快上以及“换赛道”“留痕迹”等现象。 政绩体现在自觉按规律办事上。当前，我国改革进入攻坚期和深水区，触及的利益矛盾日益复杂，阻力与挑战显著增加，拈轻怕重、回避矛盾只会无所作为，急功近利、弄虚作假必须严肃问责。“为官避事平生耻”，要强化“当干部就要解决问题”的责任担当，以攻坚克难、迎难而上的政治勇气，直面矛盾问题不回避，铲除顽瘴痼疾不含糊，应对风险挑战不退缩，所有规划都要实事求是，追求实实在在、没有水分的增长，奋力打开改革发展新天地。同时，面对历史遗留问题，是承担责任、认真解决，还是置之不理、推拖躲绕，最能展现一名干部的政绩观。要树立“新官要理旧账、新官善理旧账”的正确理念，既接权力又接责任，既谋新事又理旧账，推动解决一批历史遗留问题，坚决防止“旧问题”拖成了“大问题”、“踢皮球”踢寒了“百姓心”。 “十四五”即将圆满收官，第二个百年奋斗目标新征程实现良好开局。领导干部只有树立和践行正确政绩观，心系百姓、真抓实干，才能凝聚起推动高质量、可持续发展的磅礴动能，不断书写无愧于党、无愧于人民、无愧于时代的崭新篇章。
