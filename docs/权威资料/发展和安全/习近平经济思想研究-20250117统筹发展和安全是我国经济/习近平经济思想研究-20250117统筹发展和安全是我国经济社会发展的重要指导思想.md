# 统筹发展和安全是我国经济社会发展的重要指导思想

2025/01/17 习近平经济思想研究中心

中国特色社会主义进入新时代，以习近平同志为核心的党中央，结合新的发展实际，对发展理念和思路作出及时调整，形成了马克思主义政治经济学在当代中国、二十一世纪世界的最新理论成果，其中就包括统筹发展和安全理论。统筹发展和安全，增强忧患意识，做到居安思危，是我们党治国理政的一个重大原则。我们要深刻领会其演进脉络、重大意义，以更好指导实践工作。

**一、统筹发展和安全的演进脉络**

发展和安全是一个国家、一个民族生存与进步必须处理好的首要问题。我们党根据不同阶段所处的历史方位、所面临的形势任务，不断调整优化发展和安全的思路、目标和任务，为统筹发展和安全理论的形成和发展奠定了坚实基础。毛泽东同志在《论十大关系》中重点阐述经济建设和国防建设的关系，强调只有经济建设发展得更快了，国防建设才能够有更大的进步。邓小平同志提出正确处理改革、发展、稳定的关系，在维护政治和社会稳定中推进改革和发展，在推进改革和发展中实现政治和社会长期稳定。江泽民同志强调改革是动力，发展是目标，稳定是前提，要把改革的力度、发展的速度和社会可承受的程度统一起来。胡锦涛同志指出，把安全发展作为一个重要理念纳入我国社会主义现代化建设总体战略，这是我们对科学发展观认识的深化。

党的十八大以来，国内外形势发生复杂严峻变化。以习近平同志为核心的党中央统筹把握中华民族伟大复兴战略全局和世界百年未有之大变局，提出既重视发展问题，又重视安全问题，在二者并重基础上强调统筹，对发展和安全关系的认识走向成熟。2014年4月，习近平总书记在中央国家安全委员会第一次会议上首次提出总体国家安全观，并对贯彻落实总体国家安全观提出明确要求，强调发展是安全的基础，安全是发展的条件。同年11月，习近平总书记在中央外事工作会议上首次提出统筹发展安全两件大事。此后，习近平总书记在多个重要场合阐明发展和安全辩证关系，并强调要统筹发展和安全。2017年10月，党的十九大提出“统筹发展和安全，增强忧患意识，做到居安思危，是我们党治国理政的一个重大原则”。2020年10月，党的十九届五中全会首次把统筹发展和安全纳入“十四五”时期我国经济社会发展的指导思想，“十四五”规划和2035年远景目标纲要列专章作出战略部署。2022年10月，党的二十大将“统筹发展和安全”作为新时代党和国家事业发展的战略部署，并将其写入《中国共产党章程（修正案）》。2023年12月，中央经济工作会议将“必须坚持高质量发展和高水平安全良性互动，以高质量发展促进高水平安全，以高水平安全保障高质量发展，发展和安全要动态平衡、相得益彰”作为新时代做好经济工作的规律性认识之一。2024年7月，党的二十届三中全会审议通过《中共中央关于进一步全面深化改革、推进中国式现代化的决定》，将坚持系统观念，处理好发展和安全等重大关系，作为进一步全面深化改革的原则。

十多年来，我们坚持统筹发展和安全，加快推进安全体系与能力建设，国家安全发展根基持续夯实，我国经济发展平衡性、协调性、可持续性明显增强，国家经济实力、科技实力、综合国力跃上新台阶，发展的安全性主动权不断提升，为全面推进中国式现代化提供了坚实基础。

**二、坚持统筹发展和安全的重大意义**

党的十八大以来，以习近平同志为核心的党中央创造性提出统筹发展和安全理论，与新发展阶段、新发展理念、新发展格局、高质量发展等有紧密的内在逻辑，是习近平经济思想的重要组成部分。

**（一）从发展阶段看，夯实安全发展根基是应对复杂严峻国内外形势的必然要求**

新发展阶段是全面建设社会主义现代化国家、向第二个百年奋斗目标进军的阶段。进入新发展阶段，国内外环境的深刻变化既带来一系列新机遇，也带来一系列新挑战。世界百年未有之大变局加速演进，经济全球化遭遇逆流，国际环境进入动荡变革期。我国发展进入战略机遇和风险挑战并存、不确定难预料因素增多的时期，各种“黑天鹅”“灰犀牛”事件随时可能发生。我们比历史上任何时期都更接近、更有信心和能力实现中华民族伟大复兴的目标，但这绝不是轻轻松松、敲锣打鼓就能实现的。这就要求我们，要辩证认识和把握国内外大势，深刻认识我国社会主要矛盾发展变化带来的新特征新要求，深刻认识错综复杂的国际环境带来的新矛盾新挑战，增强机遇意识和风险意识，准确识变、科学应变、主动求变，勇于开顶风船，善于转危为机，创造更加安全的高质量发展环境。

**（二）从发展理念看，安全发展要贯穿国家发展各领域和全过程**

贯彻新发展理念是新时代我国发展壮大的必由之路。习近平总书记强调，推动创新发展、协调发展、绿色发展、开放发展、共享发展，前提都是国家安全、社会稳定。要从忧患意识把握新发展理念。从问题导向、忧患意识来认识把握，新发展理念蕴含着丰富的安全发展要求。比如，在创新发展方面，强调科技创新是推动发展的重要动力，也是国家安全的重要保障，要加快实现高水平科技自立自强，把发展基点放在创新上；在协调发展方面，强调从当前我国发展中不平衡、不协调、不可持续的突出问题出发，着力推动区域协调发展、城乡协调发展、物质文明和精神文明协调发展，推动经济建设和国防建设融合发展；在绿色发展方面，强调高水平保护是高质量发展的重要支撑，生态优先、绿色低碳的高质量发展只有依靠高水平保护才能实现；在开放发展方面，强调要把握好开放和安全的关系，织密织牢开放安全网，增强在对外开放环境中动态维护国家安全的本领；在共享发展方面，强调要做好维护社会和谐稳定工作，加大协调各方面利益关系的力度，推动发展成果更多更公平惠及全体人民。这就要求我们，要把安全和发展置于同等重要地位，把安全发展贯穿国家发展各领域和全过程，同经济社会发展一起谋划、一起部署。

**（三）从发展路径看，牢牢守住安全发展底线是构建新发展格局的重要前提和保障**

加快构建新发展格局，是立足实现第二个百年奋斗目标、统筹发展和安全作出的战略决策，是把握未来发展主动权的战略部署。从国际比较看，大国经济的特征都是内需为主导、内部可循环。在当前国际形势充满不稳定性不确定性的背景下，立足国内、依托国内超大规模市场优势，充分挖掘内需潜力，有利于化解外部冲击和外需下降带来的影响，也有利于在极端情况下保证我国经济基本正常运行和社会大局总体稳定。牢牢守住安全发展这条底线是构建新发展格局的重要前提和保障，也是畅通国内大循环的题中应有之义。这就要求我们，要坚持统筹发展和安全、开放和安全，通过构建新发展格局，形成强大的国内经济循环体系和稳固的基本盘，增强在对外开放环境中动态维护国家安全的本领，不断夯实我国经济发展根基、增强发展的安全性稳定性。

**（四）从发展质量看，必须实现高质量发展和高水平安全良性互动**

发展是解决我国一切问题的基础和关键，没有坚实的物质技术基础，就不可能全面建成社会主义现代化强国。国家安全是民族复兴的根基，社会稳定是国家强盛的前提。发展和安全二者犹如鸟之两翼、车之双轮，任何时候都不能偏废。实现发展和安全的动态平衡、相得益彰，关键在于统筹好两者的关系，推动发展和安全深度融合，实现高质量发展和高水平安全的良性互动，以高质量发展促进高水平安全，以高水平安全保障高质量发展。一方面，要善于运用发展成果夯实国家安全的实力基础。发展具有基础性、根本性，是解决安全问题的总钥匙，要不断增强我国经济实力、科技实力、综合国力，为国家安全、社会稳定提供坚实基础。另一方面，要善于塑造有利于经济社会发展的安全环境。深入推进国家安全思路、体制、手段创新，着力防范化解重大风险，以新安全格局保障新发展格局，努力实现更高质量、更有效率、更加公平、更可持续、更为安全的发展。

**三、当前经济领域统筹发展和安全的重点任务**

高质量发展是全面建设社会主义现代化国家的首要任务，是新时代的硬道理。坚持统筹发展和安全，必须牢牢扭住经济建设这个中心，强化国家经济安全保障，提高经济整体实力、竞争力和抵御各种冲击与威胁的能力，为全面建成社会主义现代化强国奠定坚实的物质技术基础。

**（一）完善社会主义市场经济体制，保证基本经济制度安全**

习近平总书记强调，制度稳则国家稳。必须坚持和完善社会主义基本经济制度，构建高水平社会主义市场经济体制，筑牢推动经济高质量发展的制度保障。坚持和落实“两个毫不动摇”，充分发挥市场在资源配置中的决定性作用，更好发挥政府作用，深化国资国企改革，为非公有制经济发展营造良好环境，促进各种所有制经济优势互补、共同发展。完善收入分配制度，完善促进机会公平制度机制，健全生产要素由市场评价贡献、按贡献决定报酬的机制。构建全国统一大市场，完善要素市场制度和规则，推动生产要素畅通流动、各类资源高效配置、市场潜力充分释放。完善产权保护、市场准入、公平竞争、社会信用等市场经济基础制度，健全宏观经济治理体系，依法维护社会主义市场经济秩序。

**（二）深入实施创新驱动发展战略，实现高水平科技自立自强**

习近平总书记指出，谁在创新上先行一步，谁就能拥有引领发展的主动权。必须健全新型举国体制，协同构建中国特色国家实验室体系，科学规划布局前瞻引领型、战略导向型、应用支撑型重大科技基础设施，坚决打赢关键核心技术攻坚战。统筹推进教育科技人才体制机制一体改革，实现科教兴国战略、人才强国战略、创新驱动发展战略有效联动。提高科技成果转化效率和质量，培养发展成果转化服务机构，更好发挥多层次资本市场对科技创新、科技成果转化的支持作用。强化企业创新主体地位，推动产业链上中下游、大中小企业融通创新，促进创新链产业链资金链人才链深度融合，提升国家创新体系整体效能。坚持自主创新和开放创新相结合，主动融入全球创新网络，建设具有全球竞争力的科技创新开放环境。

**（三）保障初级产品供应安全，夯实安全发展基础**

习近平总书记强调，以国内稳产保供的确定性来应对外部环境的不确定性。必须落实藏粮于地、藏粮于技战略，持续建设高标准农田，深入实施种业振兴行动，全面落实粮食安全党政同责，确保中国人的饭碗牢牢端在自己手中。立足国内多元供应保能源安全，强化煤电支撑性调节性作用，加快规划建设新型能源体系，完善新能源消纳和调控政策措施，实现煤炭供应安全兜底、油气核心需求依靠自保、电力供应稳定可靠。加强矿产资源国内勘探开发和增储上产，实施新一轮找矿突破战略行动，提高海洋资源、矿产资源开发保护水平，完善战略性矿产资源探产供储销统筹和衔接体系。加快建设大国储备体系，不断增强国家储备实力。实施全面节约战略，推进资源节约集约利用，从生产、消费、流通等各环节减少资源消耗。在立足国内前提下，广泛利用国际资源，扩大海外优质资源权益，保障国际战略通道和关键节点安全。

**（四）提升产业链供应链韧性和安全水平，加快建设现代化产业体系**

习近平总书记强调，加快建设以实体经济为支撑的现代化产业体系，关系我们在未来发展和国际竞争中赢得战略主动。必须健全因地制宜发展新质生产力体制机制，加快形成同新质生产力更相适应的生产关系，促进各类先进生产要素向发展新质生产力集聚。把发展经济的着力点放在实体经济上，健全促进数字经济和实体经济深度融合制度。扎实推进新型工业化，做强做优做大数字经济。完善发展服务业体制机制，推进生产性服务业融合发展、生活性服务业多样化发展。健全强化重点产业链发展体制机制，建立产业链供应链安全风险评估和应对机制，完善产业在国内梯度有序转移的协作机制，打造自主可控的产业链供应链。优化基础设施布局、结构、功能和系统集成，加快构建现代化基础设施体系，夯实国家发展和安全保障的基础支撑。

**（五）坚持底线思维，防范化解经济领域重大风险**

习近平总书记指出，树立底线思维，把困难估计得更充分一些，把风险思考得更深入一些，注重堵漏洞、强弱项，下好先手棋、打好主动仗，有效防范化解各类风险挑战，确保社会主义现代化事业顺利推进。必须加快完善风险源头防控、风险监测预警和早期纠正、风险事后处置等机制，用足用好政策工具箱。加快推进中小金融机构风险处置，推动积极稳妥化解房地产企业风险，促进金融与房地产良性循环，建立同高质量发展相适应的政府债务管理机制。加强重要民生商品的产销保供，持续完善重要民生商品价格调控机制，维护价格稳定。强化就业、教育、社会保障、医疗卫生、食品安全、安全生产等重点领域的风险防控，不断增强人民群众的获得感、幸福感、安全感。

**（六）统筹开放和安全，扩大高水平对外开放**

习近平总书记强调，要把握好开放和安全的关系，织密织牢开放安全网，增强在对外开放环境中动态维护国家安全的本领。必须坚定不移推进高水平对外开放，稳步扩大规则、规制、管理、标准等制度型开放，维护以世界贸易组织为核心的多边贸易体制，积极参与全球经济治理体系改革，提供更多全球公共产品。优化区域开放布局，实施自由贸易试验区提升战略，加快建设海南自由贸易港，推动共建“一带一路”高质量发展，加快打造陆海内外联动、东西双向互济的全面开放格局。深化外贸体制改革、外商投资和对外投资管理体制改革，深入推进贸易和投资自由化便利化，强化贸易政策和财税、金融、产业政策协同，打造贸易强国制度支撑和政策支持体系。强化海外利益和投资风险预警、防控、保护体制机制，不断提高海外安全保障能力和水平。
