# Kuner C, Svantesson D J B, Cate F H, et al. Transborder Data Flows and Data Privacy Law\[M]. Oxford: Oxford University Press, 2017.

Kuner C, Svantesson D J B, Cate F H, et al. Transborder Data Flows and Data Privacy Law\[M]. Oxford: Oxford University Press, 2017.

*ISBN*-13. 978-0199674619

**Christopher Kuner（克里斯托弗·库纳）是当代**国际数据保护法、跨境数据流动与数字治理领域最具影响力的法学学者之一，被普遍认为是 **“跨境数据法（international data law）”** 研究的奠基性人物之一。他长期活跃于欧洲与国际法学界，在学术研究、国际规则制定与政策咨询层面均具有重要影响。

### **一、学术定位：跨境数据流动的制度解释者**

Kuner 的核心研究问题可以概括为：

> **在数据跨越国界、算法全球运行的条件下，法律规则应如何实现有效适用与协调？**

他并不将数据保护视为单一国家的内部事务，而是明确提出：

- 数据流动天然具有**跨国性**
- 数据治理必然涉及**国际法、国际私法与国内法的交叉适用**
- 单一国家立法难以独立解决跨境数据与算法问题

这一立场为后来的**人工智能规则国际合作**提供了坚实的法理基础。

***

### **核心学术贡献一：跨境数据法（Transborder Data Law）**

在其主编的权威著作

*Transborder Data Flows and Data Privacy Law*（2017）中，Kuner 系统提出：

- 跨境数据流动已成为全球经济和数字技术运行的基础条件
- 数据保护规则在跨国适用中面临管辖冲突与制度碎片化
- 需要通过国际协调与软法机制降低制度摩擦

他由此推动形成了“**跨境数据法**”这一研究范式，使数据治理问题正式进入国际法与全球治理框架。

***

### **核心学术贡献二：对欧盟 GDPR 的法理阐释**

Kuner 是国际公认的**GDPR 权威解释者**之一，长期研究：

- GDPR 的**域外适用（extraterritoriality）**
- 欧盟数据保护规则的国际影响
- 欧盟模式在全球扩散中的法律张力

他强调，GDPR 的域外适用并非单纯的“规则输出”，而是一种通过法律规则回应数据跨境流动现实的制度尝试，其效果高度依赖于国际协调与相互承认机制。
