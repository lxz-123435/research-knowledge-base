# 开放型经济

*开放型经济*是总体制度目标体系，*高水平对外开放*是其落地执行框架；*制度型开放*和*涉外法治*是支撑机制；*数字贸易*、*服务业高水平开放*是具体开放领域；*全球产业链供应链*和*国际合作*是开放的物质基础与合作制度平台；*全球治理*是开放型经济参与构建的国际规则环境。
