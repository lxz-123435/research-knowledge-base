# 惠耕田.沟通理性与国际合作\[J].世界经济与政治,2009,(02):24-31+3-4.

\[1]惠耕田.沟通理性与国际合作\[J].世界经济与政治,2009,(02):24-31+3-4.

【内容提要 】在国际关系研究中国际合作是位于互动层次的现象。 华尔兹的结构现实主义和温特的结构建构主义分别强调物质结构和理念结构的重要性, 但 二者都追求结构性解释, 其解释力难以覆盖位于互动层次的国际合作。 新自由制度主义理论立足互动层次, 对国际合作的研究做出了很大的贡献, 而且在与新现实主义的论战中, 迫使后者承认了合作中对获益差距敏感系数 ( K) 值的条件性问题。 但由于新自由制度主义坚持个体主义的本体论和方法论, 难以充分解释K<0或 K=0的情况, 借助行为体沟通理性的假定和建构主义互动论原理可以推论出沟通有助于社会信任的建立, 从而有助于解决信息真实性问题以及相对获益对合作的干扰问题, 进而推动国际合作的持续进行。 因此, 行为体沟通理性的引入 或许可以为国际合作的研究开辟新的学术空间。

【关键词】  沟通理性;国际合作;互动层次;博弈论

国际合作是国际关系理论研究中的热点问题。①根据合作产生的途径, 米尔纳把合作分为三种类型:心照不宣的合作 (  tacitcooperation)、经谈判导致的合作 (  negotiated cooperation)和强制的合作 (  mi- posed c **o**peration)。 对于国家来说, 无政府状态下的合作可以减缓安全困境的压力, 使得国家在混合动机博弈中 ( mixed-motive game)取得双赢。② 这种博弈指的是一种相互依赖, 其中冲突与和谐两种利益都体现得很充分。 在这种博弈中, 背叛与合作的动机都存在, 囚徒困境是混合动机博弈的典型。 因此国际合作是一种值得珍视的价值, ③对其进行关注和研究的努力一直持续着。 西方关于国际合作的研究已经取得了一些成果, 围绕着无政府状态下国家间合作的可能性与面临的障碍、国际合作的动力与国际合作的条件, 不同的理论范式有不同的探索和回答。 中国学者关于国际合作的理论研究和经验研究也取得了一些成果。①  本文旨在从行为体的沟通理性出发, 从社会信任的角度探索合作的可能性。该项研究力图融合理性选择模式和建构主义互动论的基本原理对国际合作现象进行探索。 本文第一部分主要介绍社会行动的分类和沟通理性, 重点突出策略行动和沟通行动以及在沟通理性的作用下, 前者在互动中向后者的转化。 第二部分详细梳理主流理论范式关于国际合作现象的解释以及沟通理性可能的贡献。 最后本文将对沟通理性在国际合作应用研究中的可行性做一个展望。

一社会行动的分类与沟通理性

马克斯 · 韦伯认为社会行动的本质是以他人举止为取向的个人行动, 强调社会行动具有针对他人的主观意义 ( 主观动机 )。 只有具有主观意义的社会行动才是可以理解的, 因此人的行动 (  action)与动物的刺激反应行为 (  behavior)是有区别的。 根据合理性从高到低的程度, 韦伯把社会行动分为四种类型, 即目的合理性行动、价值合理性行动、情感行动和传统行动。②  在目的合理性行动中, 行为体遵循的是后果性逻辑, ③行为体对于行动的目的、价值以及达到目的的手段和后果都有清醒的认识和自觉的选择。 韦伯也谈到价值理性, 但价值合理性行动依据对固有价值的纯粹信仰, 并不考虑行动后果, 因此其理性程度不如工具理性高。 情感行动由现时的情绪或情感状况决定, 不考虑价值和后果。 传统行动由约定俗成的习惯决定, 对于行动的目的、价值和后果都不考虑。 传统行动和情感行动都是没有经过理性思考的行动。 尽管韦伯的目的行动中包含有价值的考虑, 但是其理性程度的高低仍然以预期后果是否实现来判断。 所以韦伯所说的合理性更多的是一种工具理性, 理性选择具有工具性质, 其过程完全受行动结果的支配。 行动不是因为其自身而受到珍视或选择, 行动的价值评判取决于其实现目的的有效性。④

哈贝马斯认为韦伯的社会行动以追求目的、取向成功为主, 其框架过于狭隘, 因此其理性概念也过于褊狭。 哈贝马斯提出了目的行动、规范调节行动、戏剧性行动和沟通行动四种社会行动类型。 在目的行动中, 如果把其他至少一个同样具有目的行为倾向的行为体对决定的期待列入对自己行为效果的计算范围, 那么这种目的行动也就发展成为了策略行动。⑤  这样哈贝马斯就在微观层面细化了韦伯的目的行动, 引入了博弈的视角。 规范调节行动 ( 循规行动 ) 是依据群体共同价值为取向的行动, 行动以角色身份为基础, 遵循适当性逻辑, ⑥这种行动要求行为体必须对某种社会规范达到很高的内化程度。行为体这么做完全出于理所当然, 并不对行为意义进行反思。 可见循规行动的真正原因在于社会结构, 是社会规范激活了具体情境下行为体的角色身份, 这是规范高度社会化的结果。 戏剧性行动旨在在观众或公众面前进行有意识的操作, 这种行为具有一定的社会性。 在沟通行动中, 行为体使用语言与非语言的符号作为理解它们彼此的情景和它们各自行动计划的手段, 以便使它们能够在如何协调自身行为上达成一致。 如果把互动理解为互动双方或多方有意识地进行行为选择, 行为体的施动性就不能被压抑, 这样的互动才是有意义的互动。 循规行动的动力源于社会结构, ①在具体情境下行为体虽然通过有意识的评估, 确定了自身具体的角色身份,但无法进行行为选择, 其施动性受到压抑, 因此循规行动不属于严格意义的互动。 戏剧性行动由于存在有意识的操作, 被操作的一方很难获取真实的信息,也很难说属于严格意义的互动。 因此在哈贝马斯的社会行动分类框架中, 如果从互动的角度来分析, 策略行动和沟通行动具有更大的研究意义和价值。

策略行动和沟通行动都是社会行动的理想类型, 前者以达到目的为导向, 后者以达致理解为导向。 行为体在策略行动中表现出工具理性, 其行为遵守预期后果逻辑。 策略行动的模型奠定了国际关系研究中博弈论的基础。 在行为体的身份和偏好预先给定的前提下, 采用理性选择模型研究战略互动的结果, 这种研究在方法论上是个体主义的。 主流国际关系理论中的新现实主义和新自由制度主义都采用了理性主义的研究设计, 对身份以及利益的来源不做探讨, 只研究其互动结果。 对于新现实主义来说, 互动结果取决于权力结构。 对于新自由制度主义来说, 互动的结果取决于国际制度。 但是新功利主义 ( 新现实主义和新自由制度主义 ) 的研究面临很大挑战, ②博弈论假定参与博弈的各方必须有共同知识, 但对于共同知识的来源, 理性主义者却很难给出满意的解释。③ 在策略行动中, 博弈中的信息交换至关重要, 行为体可以根据新的信息调整自身的战略实现预先确定的目标, 但行为体的目标本身并不会变化, 行为体在互动中只从事简单习得。当在互动中出现或明或暗的道义诉求, 或者通过谈判进行复杂习得④进而导致预先确定的目标出现调整时, 策略互动的模型要么束手无策, 不知该如何解释这些现象, 要么随意添加临时假定寻求解释。⑤而在现实经验中, 谈判中改变利益主张则是常有的事情。 更为关键的是, 对于信息传递的过程中如何确保其他博弈方确信信息的真实性, ⑥博弈论研究并未做出有力的回答。 假如没有最低程度的社会信任存在, 博弈根本就不可能产生。 这些重要的问题并不能不加质疑地放入预先假定中。

鉴于以博弈论为基础的策略行动对于很多问题无法给予满意的解释, 有必要尝试将研究对象转移到沟通行动上, 把合作理解为沟通行动。 事实上, 在现实生活中, 很多互动以策略行动开始, 以沟通行动结束, 或者说策略行动难以避免地转变为沟通行动。因为缺乏对方的合作, 单方收益最大化的目标就难以实现, 这是囚徒困境博弈的本质。 既然行动要合作完成, 必然就会有关于收益分配合理性关切的相互谅解。 这些都需要通过言语沟通完成。⑦  沟通行动中行为体体现出沟通理性。 沟通理性是一种对话式的理性, 预设至少存在两个行为体, 站在平等的立场上, 互相沟通、讨论, 希望获致相互理解和共识以
