# 当前国际经贸斗争形势下对外投资新挑战、新走势

问题——**（七）我海外综合服务保障能力尚有不足**

**服务平台功能碎片化。公共信息来源分散，** 现有的公共服务产品分散在与外经贸有关的各个部门，尚未构建跨部门的整合机制。政策措施、国别信息、法律咨询、金融资源、风险预警等信息存在于多个不同的海外服务平台，企业在实际应用中难以高效获取。**信息动态更新渠道不畅，** 部分发展中国家产业政策、法律环境、税收规则等变动较快，但目前缺乏覆盖多领域的综合性动态更新渠道，常导致企业投资决策滞后或失误。例如，资源民族主义政策频发的国家（如印尼、刚果金等）财税条款多变，但企业难以及时通过有效渠道获取政策变动信息，无法提前调整投资策略、优化成本核算，造成项目预期收益缩水，甚至面临额外税负压力。

**专业服务产品质量需进一步提升**。**国内专业服务机构海外布局滞后**，如律所、金融机构等国际化程度低，在海外的分支机构数量少、运营困难，难以提供及时有效的服务。**专业机构服务能力参差不齐**，且缺乏有效整合，企业需自行筛选不同领域的服务商，难以获得全链条解决方案。**跨境融资服务不能满足企业需求**，我国对出海企业提供的跨境融资服务分散于银行、政策性金融机构等不同主体，产品设计缺乏针对性。‌政策性银行和商业金融机构的海外信贷产品多集中于大型项目，中小企业的融资需求难以满足，且汇率避险工具普及率低。**合规与法律支持方面相对薄弱**，企业面临东道国本地化要求、环保标准等合规挑战，但专业法律咨询和纠纷解决服务覆盖有限。

举措——**（五）完善海外综合服务体系**

**升级线上线下服务平台**\*\*。**各地方要因地制宜**建设本地区“线上+\*\***线下”出海服务平台**，汇集政府、专业机构、行业组织等境内外服务资源，成为企业出海获取基础性信息的“一站式”窗口。要依托地方政务服务中心，**持续优化业务办理流程，** 推广使用对外投资电子证照，推动政务数据共享，提高服务效率。要集聚企业出海全链条服务需求，系统推进法律、仲裁、知识产权、财务会计、安全保护、金融保险、人力资源等领域优质专业服务机构整合，**培育富有地域特点的专业服务品牌**。

**探索创新出海服务模式。要结合各地方实际打造出海综合服务港**，依托自由贸易试验区、自由贸易港、服务业扩大开放综合示范区等开放平台，聚集开放枢纽、总部经济、服务资源等优势，主动对接高标准国际经贸规则，探索对外投资服务领域改革，帮助企业匹配出海服务资源。各地方要以适当形式推介各领域、各层级优质服务资源，打破信息壁垒，帮助企业**有效识别和精准匹配出海服务资源**。长三角、京津冀、粤港澳大湾区、长江经济带等地区要发挥比较优势，**高效整合优质服务资源**，发挥互补协同效应，提升海外综合服务水平。依托港澳在国际投资贸易、金融、法律等领域优势，共同增强企业出海服务能力。

**优化境外商会服务功能。继续**推动各地方的境外商（协）会积极加入境外中资企业商（协）会联席会议，扩大联席会议成员单位规模，形成面向全球境外中资商会的服务网络，提升联席会议的覆盖面和影响力。借助联席会议的机制平台，维护与表达中资企业利益和诉求，推进所在国投资营商环境改善，提升企业国际化经营水平。**支持各地方境外中资企业商协会设立行业分会、区域性分会和专业委员会**，增强协会成员业务沟通联络、信息资源共享，为出海企业更具针对性、细致性的属地支持。

**提升法律、财会、咨询等服务水平。提升各地方**律师事务所、仲裁机构国际化服务能力，培训国际一流律师事务所和仲裁机构。支持各地方**会计师事务所**为出海企业提供审计、评估、尽职调查、财务税务顾问等服务，拓展审计鉴证、管理咨询、供应链管理等跨境业务。支持**咨询公司**为出海企业提供全球发展战略统筹、公司治理、营销设计、品牌推广等服务，助力企业向价值链高端攀升。鼓励**法律、财会、咨询机构**发挥知识密集优势，整合关联行业资源，延伸服务链条，拓展服务领域，为企业出海提供一站式、系统性解决方案。

**增强金融机构跨境服务创新能力。鼓励各地方**商业性金融机构创新完善服务产品和服务方式，拓展企业融资渠道。支持金融机构完善海外布局发展，为企业提供定制化、“跟随式”服务。鼓励**保险机构**托大承保规模和覆盖面，开发适合出海企业需求的险种，积极开展再保险业务，支持专业保险经纪公司参与企业出海服务。

**加强海外业务技术支持能力。支持各地方**信息技术等领域服务机构创新服务方式，为出海企业进行数字赋能。鼓励**知识产权、标准检验认证、人力资源等服务机构**提升国际化服务能力，积极参与相关国际标准和规则制定，开展国际交流合作。支持**有关专家智库**建设，加强国别行业发展研究分析，为企业提供决策咨询和智力支撑。
