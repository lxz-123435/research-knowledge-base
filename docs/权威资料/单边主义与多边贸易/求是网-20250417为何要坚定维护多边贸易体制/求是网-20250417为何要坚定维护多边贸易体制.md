# 为何要坚定维护多边贸易体制

求是网评论员 \[求是网]\(javascript:void$0$;) *2025年4月17日 22:59* *北京*

近期，美国政府对包括中国在内的所有贸易伙伴实施所谓“对等关税”。这种以极限施压谋取单边利益的做法，严重违反世界贸易组织规则，严重损害以规则为基础的多边贸易体制，给世界经济带来更多不确定性不稳定性。

以世界贸易组织为核心的多边贸易体制是全球贸易的根本制度保障，以最惠国待遇原则为基础的多边关税承诺确保所有成员在透明、可预期和非歧视的环境下开展贸易和合作。这套行之有效的规则是包括美国在内的成员基于对上世纪30年代以邻为壑贸易战造成的惨痛后果深刻反思建立的，是自1947年以来多边贸易谈判达成的成果，也是第二次世界大战后世界经济繁荣稳定的基石。多边贸易体制来之不易，是全体成员的公共产品，任何成员都无权破坏。

美国借由在国际贸易中吃了亏，单方面实施所谓“对等关税”，本质上是追求“美国优先”、“美国特殊”。相关专家认为，根据世界贸易组织最惠国待遇原则，美国所谓“对等关税”至少在两方面违反世界贸易组织规则：一是对不同贸易伙伴施加差异化关税，违反成员间互不歧视的承诺；二是将关税税率调至超出与其他成员协商约定的上限，构成违规行为。美国以一己之私公然践踏世界贸易组织规则，罔顾多边贸易谈判达成的利益平衡结果，给多边贸易体制带来严重冲击，遭到国际社会强烈反对。在近日召开的世界贸易组织货物贸易理事会会议上，欧盟、英国、加拿大、日本、韩国、马来西亚、巴西等46个世界贸易组织成员强烈要求美国遵守规则、信守承诺。

中国作为负责任大国，始终是多边贸易体制的坚定维护者。贸易战、关税战没有赢家，保护主义没有出路。中方不愿打、但也不怕打，绝不会坐视中国人民的正当权益被剥夺，绝不会坐视国际经贸规则和多边贸易体制被破坏。在美国公然冒天下之大不韪，侵害国际社会公利的当下，中国第一时间坚决反制美方霸凌行径，是捍卫真正的多边主义、维护多边贸易体制的必要之举。

开放合作是历史潮流，互利共赢是人心所向。谁是多边贸易体制的破坏者、谁是建设者？谁搞歪门邪道、谁行大道正道？谁在开历史倒车、谁站在历史正确的一边？答案一清二楚。世界要公道，不要霸道！中方愿与各方一道，共同维护开放、透明、公正、包容、非歧视的多边贸易体制，推动经济全球化朝着正确方向发展。
