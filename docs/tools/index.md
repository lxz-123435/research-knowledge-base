# 研究工具

研究过程中使用的工具和方法。

## 自动化脚本

### 📝 AI分析
- [AI资料分析](ai_analyze.py) - 自动解析PDF、生成摘要

### 🚀 部署
- [自动部署](deploy.sh) - 一键部署网站

## 使用方法

### 分析PDF文件
```bash
python3 scripts/ai_analyze.py /path/to/file.pdf
```

### 批量分析
```bash
python3 scripts/ai_analyze.py /path/to/directory/
```

### 部署网站
```bash
bash scripts/deploy.sh
```
