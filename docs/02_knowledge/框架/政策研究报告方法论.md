# 政策研究报告写作方法论

> 三步法 + 四层逻辑 + 两个对齐

---

## 一、三步法：回顾—判断—对策

绝大多数政策研究报告结构：

```text
第一步：回顾过去
第二步：判断形势
第三步：提出对策
```

对应结构：

```text
第一章 现状/回顾
第二章 风险/趋势
第三章 对策/建议
```

| 功能 | 内容 |
| -- | ----------- |
| 回顾 | 投资情况 |
| 判断 | 风险形势 |
| 对策 | 管理建议 |

---

## 二、四层逻辑：宏观 → 政治 → 制度 → 企业

风险分析层级：

```text
宏观环境风险
政治安全风险
制度政策风险
企业经营风险
```

| 风险类型 | 说明 |
| ---- | ------- |
| 宏观 | 全球经济不确定 |
| 经济结构 | 经济格局重构 |
| 政治 | 地缘政治 |
| 安全 | 地区安全 |
| 制度 | 贸易保护 |
| 监管 | 合规风险 |
| 社会 | 社会文化 |
| 经营 | 供应链 |

---

## 三、风险框架构建方法

### 方法1：PEST框架

```
P 政治
E 经济
S 社会
T 技术
```

扩展为：政治、经济、社会、制度、产业

### 方法2：产业链框架

```
资源 → 产业 → 市场 → 金融
```

### 方法3：企业经营框架

```
市场风险、运营风险、政策风险、安全风险
```

---

## 四、对策设计方法

遵循原则：**风险 → 对策**

```text
识别风险
提出能力
构建机制
提供工具
```

| 类型 | 内容 |
| -- | ------ |
| 能力 | 调研能力 |
| 机制 | 风险管理体系 |
| 工具 | 投融资工具 |
| 治理 | 合规与ESG |

---

## 五、两个对齐

### 1. 风险—对策对齐

| 风险 | 对策 |
| ----- | ----- |
| 经济不确定 | 宏观研判 |
| 地缘政治 | 风险管理 |
| 贸易保护 | 供应链布局 |
| 合规 | 合规体系 |
| 社会文化 | 本地化 |

### 2. 过去—未来对齐

第一章回顾的风险，要能在第二章延续。

---

## 六、好框架的五个标准

1. **完整性** - 覆盖主要问题
2. **层级清晰** - 宏观 → 微观
3. **逻辑递进** - 回顾 → 判断 → 对策
4. **对应关系** - 风险 → 对策
5. **可写性** - 每个标题能写2000字

---

## 七、通用模板

```text
第一章 现状与特点
  规模
  结构
  特征

第二章 风险与挑战
  宏观环境
  政治安全
  制度政策
  企业经营

第三章 对策建议
  提升能力
  完善机制
  优化布局
  强化保障
```

---

## 八、核心方法论

> 用结构理解问题，用风险组织内容，用对策形成闭环。

---

*方法论来源：国研中心、社科院、商务部研究院通用框架*
