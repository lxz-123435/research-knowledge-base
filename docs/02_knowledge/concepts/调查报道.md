# 调查报道

## 定义

调查报道是新闻记者通过深入调查、采访和研究，揭露事实真相的新闻报道方式。

## 核心特征

| 特征 | 说明 |
|------|------|
| 深度 | 长时间调查取证 |
| 真相 | 揭露隐蔽事实 |
| 风险 | 可能面临压力甚至危险 |

## 来源

- 《从不妥协：法拉奇传》

## 关键要素

1. **信息源开发** - 建立可靠线人网络
2. **交叉验证** - 多源信息核实
3. **风险评估** - 预判报道后果

## 典型案例

- 水门事件（伍德沃德 & Bernstein）
- 越南战争报道（法拉奇）
- 中东冲突报道（法拉奇）

## 相关人物

- 伍德沃德
- 卡帕
