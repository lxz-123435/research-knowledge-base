# 时代选择

## 定义

时代选择是指个人在特定历史情境下做出的价值判断和行动决策。

## 核心内涵

```
个人选择 = 时代背景 + 价值立场 + 行动决策
```

## 来源

- 《从不妥协：法拉奇传》

## 核心要素

| 要素 | 说明 |
|------|------|
| 时代背景 | 理解历史情境 |
| 价值立场 | 明确价值判断 |
| 行动决策 | 做出实际选择 |

## 法拉奇的选择

在冷战、反殖民运动、思想激荡的时代：

- 选择成为战地记者
- 选择直面权力
- 选择追问真相

> 这不仅是职业选择，更是价值选择。

## 当代意义

### 时代特征

- 技术变革迅速
- 信息环境复杂
- 社会矛盾多变

### 个人应对

1. 理解时代 - 在理解背景基础上判断
2. 保持独立 - 在不确定中保持人格独立
3. 勇于发声 - 在公共问题上表达观点
4. 主动选择 - 用个人选择回应时代
