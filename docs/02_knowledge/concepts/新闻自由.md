# 新闻自由

## 定义

新闻自由是指新闻媒体和新闻工作者在法律范围内，进行新闻报道和言论的自由权利。

## 核心要素

| 要素 | 说明 |
|------|------|
| 采访自由 | 获取信息的权利 |
| 出版自由 | 发布信息的权利 |
| 评论自由 | 发表观点的权利 |

## 来源

- 《从不妥协：法拉奇传》

## 相关理论

- 第四权力理论 - 媒体作为三权之外的监督力量
- 公共利益理论 - 新闻自由服务于公共利益

## 关键议题

1. 知情权 - 公众获取信息的权利
2. 监督权 - 媒体监督政府和权力的责任
3. 隐私权 - 个人隐私与公共利益的平衡

## 典型人物

- 奥莉娅娜·法拉奇 - 用一生践行新闻自由
- 普利策 - 调查报道奠基人
