# 访谈研究方法

## 定义

访谈研究方法是通过与研究对象进行深度对话，获取一手资料的研究方法。

## 核心要素

### 1. 访谈准备

```
确定目标 → 设计问题 → 选择对象 → 预约时间
```

### 2. 访谈技巧

| 技巧 | 说明 |
|------|------|
| 开放式问题 | 引导深入回答 |
| 追问 | 挖掘细节 |
| 沉默 | 给对方思考时间 |
| 观察 | 注意非语言信息 |

### 3. 访谈类型

- **结构化访谈** - 预设问题，严格执行
- **半结构化访谈** - 灵活调整
- **深度访谈** - 长时间、开放性

## 来源

- 《从不妥协：法拉奇传》

## 法拉奇访谈模型

### 三步法

```
第一步：打破对方话术
    ↓
第二步：用简单问题直指核心
    ↓
第三步：持续追问，不退缩
```

### 核心特征

| 特征 | 说明 |
|------|------|
| 攻击性 | 不畏惧权力 |
| 求真相 | 打破官方叙事 |
| 情感投入 | 用生命去采访 |

## 应用场景

- 人物传记
- 新闻采访
- 学术研究
- 口述历史

## 注意事项

1. 伦理合规 - 知情同意
2. 记录完整 - 录音/笔记
3. 客观呈现 - 不带预设
