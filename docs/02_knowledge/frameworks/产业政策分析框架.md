# 产业政策分析框架

> 基于IMF研究的分析框架

## 理论来源

- IMF "新产业政策观察"（NIPO）方法论
- 大语言模型政策动机识别

## 核心理论

产业政策分析需要关注：
1. 政策动机
2. 政策工具
3. 政策目标
4. 政策演变

## 分析维度

| 维度 | 说明 |
|------|------|
| 政策动机 | 为什么出台政策？ |
| 政策工具 | 用什么手段？ |
| 政策目标 | 指向什么领域？ |
| 政策演变 | 如何随时间变化？ |

## 适用问题

- 各国产业政策比较
- 产业政策趋势分析
- 政策效果评估

## 分析路径

```
背景分析 → 政策动机 → 政策工具 → 政策目标 → 效果评估
```

## 应用案例

### 案例：金融危机后产业政策演变

- 2009-2019：传统产业政策（竞争力、气候）
- 2020后：新产业政策（供应链、安全、地缘政治）
