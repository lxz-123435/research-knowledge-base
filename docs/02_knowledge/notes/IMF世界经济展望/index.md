# IMF 世界经济展望 2025年7月

## 核心观点

**主题：持久不确定中的脆弱韧性**

全球经济增长预计2025年为3.0%，2026年为3.1%。2025年预测比4月份的预测上调0.2个百分点。

## 关键数据

| 指标 | 2025年 | 2026年 |
|------|--------|--------|
| 全球经济增长 | 3.0% | 3.1% |
| 全球通胀 | 4.2% | 3.6% |

## 主要发现

### 1. 增长韧性超预期
- 关税预期前的提前抢购强于预期
- 平均有效美国关税率低于4月宣布的水平
- 金融条件改善（包括美元走弱）
- 一些主要司法管辖区的财政扩张

### 2. 通胀走势
- 整体通胀路径与4月预测相似
- 国家间差异显著
- 美国通胀预计保持高于目标
- 其他大型经济体通胀更为温和

### 3. 风险与不确定性
- **下行风险：**
  - 有效关税率回升可能导致增长减弱
  - 不确定性加剧可能开始更严重影响活动
  - 地缘政治紧张可能扰乱全球供应链
  - 财政赤字扩大或风险偏好下降可能推高长期利率

- **上行风险：**
  - 贸易谈判达成可预测框架并降低关税

### 4. 主要经济体表现
- **美国**：2025年第一季度实际GDP下降0.5%（年化），三年来首次季度收缩
- **欧元区**：GDP加速至2.5%，由投资和净出口驱动
- **中国**：实际GDP增长6.0%（年化），超出预期，主要由出口驱动
- **日本**：经济收缩0.2%（年化）

## 政策建议

1. **缓解紧张局势** - 带来信心、可预测性和可持续性
2. **保持价格和金融稳定**
3. **恢复财政缓冲**
4. **实施必要的结构性改革**
