# 中国对外直接投资统计公报 2024

## 核心观点

2024年中国对外直接投资保持增长，全年流量达1922亿美元，增长8.4%，位列全球第三。

## 关键数据

| 指标 | 数值 | 变化 |
|------|------|------|
| 对外直接投资流量 | 1922亿美元 | +8.4% |
| 对外直接投资存量 | 3.14万亿美元 | +6.2% |
| 全球排名 | 第3位 | - |
| 占全球流量比重 | 11.9% | - |
| 占全球存量比重 | 7.2% | - |

## 流量构成

| 类型 | 金额 | 占比 |
|------|------|------|
| 新增股权投资 | 730.5亿美元 | 38% |
| 收益再投资 | 778.9亿美元 | 40.5% |
| 债务工具投资 | 412.6亿美元 | 21.5% |

## 按行业分布

- **金融类**：237.8亿美元（+30.5%）
- **非金融类**：1684.2亿美元（+5.9%）

## 境外企业情况

- 境内投资者在境外设立企业：约5.2万家
- 分布国家/地区：190个
- 境外企业资产总额：超6万亿美元
- 境外企业从业人员：502.1万人
- 雇用外方员工：330.4万人（占65.8%）

## 投资带动效应

- 带动出口：2110亿美元（占同期货物出口5.9%）
- 带动进口：712亿美元（占同期货物进口2.8%）
- 境外企业销售收入：36261亿美元（+3.8%）

## 主要特点

1. **流量创历史新高**：1922亿美元是历年第二高
2. **存量稳步增长**：3.14万亿美元，仅次于美国和荷兰
3. **结构优化**：收益再投资占比提升
4. **质量提升**：带动贸易和就业效果显著
