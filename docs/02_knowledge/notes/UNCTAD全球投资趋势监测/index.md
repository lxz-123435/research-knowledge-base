# 《UNCTAD全球投资趋势监测》阅读笔记

> UNCTAD | 2024年10月

---

## ① 核心问题

本文试图回答什么问题？

1. 2024年上半年全球FDI趋势如何？
2. 绿地投资为何下降？
3. 哪些行业逆势增长？
4. 投资环境面临哪些挑战？

---

## ② 核心观点

1. **FDI微弱增长**：2024上半年仅增长1%（不含欧洲导管经济体）

2. **绿地投资下降**：
   - 数量降11%
   - 价值降4%
   - ICT项目减少是主因

3. **能源项目增长**：能源和天然气项目增加5%

4. **半导体项目亮眼**：大型半导体项目价值显著，主要集中在美国

5. **投资环境严峻**：地缘危机和经济碎片化导致企业谨慎

## ⑥ 现实启发

1. **投资谨慎**：企业在地缘危机中趋于保守

2. **行业分化**：危中有机，能源和半导体是亮点

3. **供应链重构**：供应链安全驱动投资决策

4. **政策启示**：稳定投资环境吸引外资
