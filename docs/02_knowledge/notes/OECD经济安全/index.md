# OECD 经济安全报告 2025

> 主题：变局中的经济安全 (Economic Security in a Changing World)

## 基本信息

- **发布机构**: 经济合作与发展组织 (OECD)
- **发布时间**: 2025年9月
- **系列**: New Approaches to Economic Challenges (NAEC)

---

## 核心观点

经济安全是各国和社会稳定与繁荣的关键。在当前高度互联和复杂的全球环境中，不同方面的经济安全相互交织，如供应链韧性、能源安全和网络安全。

## 主要发现

### 1. 供应链相互依赖
- 国际供应链效率与脆弱性并存
- 贸易依赖度衡量存在挑战
- 供应链去风险政策可能导致负面经济影响

### 2. 半导体产业
- 生产高度集中在亚洲
- 价值链各环节由不同经济体主导
- 需要国际合作增强韧性

### 3. 关键原材料
- 生产高度集中
- 各国政府采取措施增强供应安全
- 负责任采购日益重要

### 4. 能源安全
- 清洁转型与能源安全密切相关
- 电力安全是跨部门问题
- 关键矿物对清洁能源未来至关重要

### 5. 投资安全
- 投资筛选机制日益普及
- 审查范围不断扩大
- 交易数量显著增加

---

## 政策建议

1. **增强供应链韧性**
   - 多元化供应来源
   - 建立战略储备
   - 促进国际合作

2. **促进关键产业投资**
   - 支持国内生产
   - 建立合作伙伴关系

3. **加强能源安全**
   - 加速清洁能源转型
   - 确保能源供应多元化
   - 动员清洁能源投资

4. **完善网络安全**
   - 建立数字安全风险管理框架
   - 保护关键活动
   - 加强国际合作
