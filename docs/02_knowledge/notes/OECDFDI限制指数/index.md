# OECD 全球外资限制指数 2024

## 核心观点

**主题：外国直接投资监管限制趋势**

外资限制随时间推移有所放宽，但自由化步伐放缓。亚洲、中东和非洲显示出更大的外资改革潜力。

## 主要发现

### 1. 全球外资限制趋势
- 过去几十年，全球外国直接投资法规总体上变得不那么严格
- 但自由化步伐近年来有所放缓
- 自1997年以来，最重大的改革发生在亚洲
- 2018-2023年，中东和北非国家也取得了显著进展

### 2. 国家间差异显著
- 2023年，对FDI的监管限制在不同国家之间持续存在显著差异
- 最限制性四分位国家的FDI法规平均比最不限制性四分位国家严格23倍
- OECD国家比非OECD国家大约不严格3.2倍

### 3. 限制类型
- 外国股权限制对外资限制的贡献最大
- 初级部门和服务业的FDI限制仍然特别重要

### 4. 投资筛选机制
- 新增或修订的投资筛选机制数量正在增加
- 各国越来越重视国家安全审查

## 政策含义

1. **亚洲、中东和非洲显示出更大的外资改革潜力**
2. **投资筛选机制正在成为新的政策工具**
3. **各国越来越关注国家安全因素**
