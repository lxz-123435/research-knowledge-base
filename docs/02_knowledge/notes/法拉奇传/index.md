# 《从不妥协：法拉奇传》阅读笔记

> 作者：克里斯蒂娜·德·斯特凡诺 | 出版社：广西师范大学出版社

---

## 核心问题

这本书试图回答什么问题？

1. 为什么法拉奇能成为20世纪最传奇的女记者？
2. 她的"从不妥协"从何而来？
3. 一个女性如何突破社会结构限制，成为世界级战地记者？

---

## 核心观点

1. 奥莉娅娜·法拉奇是20世纪最伟大的女性记者之一，她用笔和勇气对抗权力
2. 她的"从不妥协"源于童年的苦难经历——法西斯压迫、反法西斯家庭
3. 新闻采访对她而言是一场"终极冒险"，她用生命去追寻真相
4. 她的职业生涯本身就是一部女性 empowerment（赋权）的传奇
