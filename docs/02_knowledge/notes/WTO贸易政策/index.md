## 核心观点

WTO贸易监测报告显示，在地缘政治紧张和气候变化危机加剧的背景下，越来越多的内向型和单边贸易政策决策给世界经济带来不确定性。

## 主要发现

### 1. 贸易限制措施激增
- 与2023年11月报告相比，本期报告期间新增贸易限制措施覆盖金额大幅增加（从3371亿美元增至8876亿美元）
- 进口限制存量自2009年以来稳步增长

### 2. 贸易救济增加
- WTO成员启动贸易救济调查的频率增加（每月28.2件，上期16.7件）

### 3. 出口限制上升
- 自2020年以来出口限制数量增加
- 首先是疫情背景，随后是乌克兰战争和粮食安全危机

### 4. 国家安全考量
- 几个经济体宣布并实施了以国家安全为由的贸易措施
- 覆盖金额约796亿美元，占世界贸易0.2%

### 5. 服务贸易
- 期间成员引入了134项新措施，大部分为贸易便利化

### 6. 气候相关措施
- 许多政府推出了气候相关措施、环境影响减量项目和向低碳经济转型的计划

### 7. 贸易碎片化
- 越来越多的贸易在志同道合的经济体之间进行
- 这一趋势因乌克兰战争而加速
