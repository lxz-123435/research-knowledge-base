# 产业政策范式转移：从效率到安全

> 思想洞见 · 来源：IMF Working Paper

---

## 核心观点

> 产业政策的核心逻辑正在发生范式转移：从"效率优先"转向"安全优先"。

---

## 逻辑分析

### 推理链条

1. **过去**：全球化时代，产业政策以效率和比较优势为导向
2. **现在**：地缘政治紧张，产业政策以安全和韧性为导向
3. **结果**：政策范围超出传统比较优势领域

### 详细分析

**传统产业政策逻辑**：
```
比较优势 → 专业化 → 效率最大化
```

**新产业政策逻辑**：
```
国家安全 → 供应链韧性 → 自主可控
```
