# 全球FDI趋势：分化与重构

> 思想洞见 · 来源：UNCTAD + 中国统计公报

---

## 核心观点

> 全球外国直接投资正在经历结构性分化：传统FDI增长乏力，但特定行业（能源、半导体）和新兴市场保持韧性。

---

## 逻辑分析

### 全球层面

**趋势1：增长放缓**
- 2024上半年全球FDI仅增长1%
- 绿地投资下降11%
- 原因：地缘危机、经济碎片化

**趋势2：行业分化**
- ICT投资下降
- 能源项目增长5%
- 半导体项目创新高

### 中国层面

**趋势1：持续增长**
- ODI保持增长态势
- 流量存量创新高

**趋势2：多元化**
- 行业分布多元
- 区域覆盖广泛
