# 产业链分析框架

## 框架定义

产业链分析框架是用于研究产业内各环节之间关系的系统性分析方法。

## 分析维度

### 1. 价值链分析

```
研发 → 生产 → 物流 → 营销 → 服务
```

### 2. 供应链分析

```
原材料 → 零部件 → 成品 → 配送 → 终端
```

### 3. 创新链分析

```
基础研究 → 应用研究 → 开发 → 产业化
```

## 核心指标

| 指标 | 说明 |
|------|------|
| 附加值 | 各环节价值贡献 |
| 集中度 | 市场结构 |
| 依存度 | 环节依赖程度 |
| 完整性 | 产业链完备程度 |

## 应用场景

- 产业竞争力评估
- 供应链安全分析
- 招商引资策略
- 产业政策制定

## 典型案例

- 新能源汽车产业链
- 半导体产业链
- 人工智能产业链
