# 访谈研究方法

## 框架定义

访谈研究方法是通过与研究对象进行深度对话，获取一手资料的研究方法。

## 核心要素

### 1. 访谈准备

```
确定目标 → 设计问题 → 选择对象 → 预约时间
```

### 2. 访谈技巧

| 技巧 | 说明 |
|------|------|
| 开放式问题 | 引导深入回答 |
| 追问 | 挖掘细节 |
| 沉默 | 给对方思考时间 |
| 观察 | 注意非语言信息 |

### 3. 访谈类型

- **结构化访谈** - 预设问题，严格执行
- **半结构化访谈** - 灵活调整
- **深度访谈** - 长时间、开放性

## 经典案例：法拉奇访谈

法拉奇的访谈方法论：

1. **准备充分** - 深入了解受访者背景
2. **直接提问** - 不回避尖锐问题
3. **情感投入** - 用生命去采访
4. **坚持真相** - 不接受官方叙事

## 应用场景

- 人物传记
- 新闻采访
- 学术研究
- 口述历史

## 注意事项

1. 伦理合规 - 知情同意
2. 记录完整 - 录音/笔记
3. 客观呈现 - 不带预设
