# 法拉奇采访方法论

## 核心方法

### 三步访谈法

```
第一步：打破对方话术
    ↓
第二步：用简单问题直指核心
    ↓
第三步：持续追问，不退缩
```

## 方法详解

### 1. 准备充分

- 深入研究受访者背景
- 了解其政治立场、历史
- 预设问题，但灵活调整

### 2. 攻击性提问

特征：
- 不接受官方叙事
- 追问敏感话题
- 打破砂锅问到底

### 3. 情感投入

- 用生命去采访
- 投入个人情感
- 但保持客观

## 经典案例

### 采访基辛格

## 方法论本质

| 特征 | 说明 |
|------|------|
| 强对抗 | 不畏惧权力 |
| 求真相 | 打破官方叙事 |
| 个人风格 | 独特采访艺术 |

## 与传统采访的区别

| 传统采访 | 法拉奇模式 |
|----------|------------|
| 客套问答 | 深度质询 |
| 接受官方回答 | 追问真相 |
| 保持距离 | 情感投入 |

## 适用场景

- 政治人物访谈
- 敏感话题采访
- 调查报道
