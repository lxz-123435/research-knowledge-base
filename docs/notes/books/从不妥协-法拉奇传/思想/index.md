# 法拉奇新闻思想

## 核心思想

### 1. 记者使命

> "我的工作就是提出问题，揭开真相。"

**记者不是记录者，而是提问者。**

### 2. 权力观

> "我没有朋友，只有敌人。"

权力需要被质疑，记者的职责是挑战权威。

### 3. 真相高于一切

- 真相比安全更重要
- 为真相可以冒险
- 不接受任何隐瞒

### 4. 女性主义

- 用专业而非性别赢得尊重
- 在男性主导行业突破
- 女性可以像男性一样强硬

## 对新闻业的启示

### 1. 采访是战争

> "采访不是聊天，是战争。"

不是温和对话，而是真相争夺。

### 2. 问题即武器

简单问题直指核心：
- 你为什么做这个决定？
- 你真的相信这个吗？
- 你不后悔吗？

### 3. 勇气即专业

专业不仅是技能，更是勇气。
